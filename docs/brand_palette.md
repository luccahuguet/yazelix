# Brand Palette

Yazelix brand colors should be derived from the shipped logo assets, not guessed from the terminal welcome screen.

Authoritative asset:
- [`assets/logo.png`](/home/lucca/.config/yazelix/assets/logo.png)

Observed palette from the current logo:
- `red`
- `green`
- `yellow`
- `blue`
- `purple`

Practical usage guidance:
- Use the full five-color cycle for the `YAZELIX` wordmark when a multicolor treatment is wanted.
- Prefer `blue`, `purple`, `yellow`, and `green` for readable body text and accents.
- Use `red` sparingly as a logo/brand accent rather than a default body-text color.
- Treat `yellow` and `green` as stronger highlight colors.
- Treat `purple` as the softer brand/footer accent.
- For longer welcome-card copy, prefer alternating readable base lines instead of rainbow lines:
  blue lines can use green highlights; yellow lines can use blue highlights.

Welcome-card mapping:
- `YAZELIX`: `red -> green -> yellow -> blue -> purple`
- Body lines alternate `blue` and `yellow`
- `blue` lines use `green` highlights
- `yellow` lines use `blue` highlights
- Footer base: `yellow`
- Footer accent: `yazelix` in `blue`
